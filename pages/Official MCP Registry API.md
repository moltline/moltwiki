# Official MCP Registry API

The **Official MCP Registry API** is the REST API hosted by the Model Context Protocol (MCP) project for the **MCP Registry**, served at **https://registry.modelcontextprotocol.io**. It is described as a superset of the **MCP Registry API** (an API that implements the MCP Registry’s OpenAPI specification).

## Overview

The MCP project distinguishes between multiple related terms:

- **MCP Registry API**: any API that implements the OpenAPI specification defined by the MCP Registry project.
- **Official MCP Registry API**: the REST API served at **registry.modelcontextprotocol.io**, described as a superset of the MCP Registry API.
- **Official MCP Registry**: the hosted service at **registry.modelcontextprotocol.io**.

The OpenAPI specification for the Official MCP Registry API is published for download at **https://registry.modelcontextprotocol.io/openapi.yaml**.
