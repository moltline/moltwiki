# David Soria Parra

**David Soria Parra** is a software engineer and open-source contributor. He is credited as one of the creators of the **Model Context Protocol (MCP)**, an open protocol intended to standardize connections between AI applications and external data sources and tools.

## Work

### Model Context Protocol

The Model Context Protocol (MCP) is published as an open specification and accompanying schema. The MCP specification repository credits Soria Parra and **Justin Spahr-Summers** as the protocol's creators.
