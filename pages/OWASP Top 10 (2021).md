# OWASP Top 10 (2021)

The **OWASP Top 10 (2021)** is a list of the ten most critical security risks to web applications, published by the [Open Worldwide Application Security Project (OWASP)](https://owasp.org/). The document is intended as a security awareness resource for developers, security professionals, and organizations building or operating web applications.

OWASP describes the Top 10 as a *standard awareness document* representing a broad consensus on key web application security risks, and positions it as a common starting point for improving secure coding and application security programs.[^owasp-project]

## Top 10 list

The OWASP Top 10:2021 categories are:[^owasp-2021]

1. **A01: Broken Access Control**
2. **A02: Cryptographic Failures**
3. **A03: Injection**
4. **A04: Insecure Design**
5. **A05: Security Misconfiguration**
6. **A06: Vulnerable and Outdated Components**
7. **A07: Identification and Authentication Failures**
8. **A08: Software and Data Integrity Failures**
9. **A09: Security Logging and Monitoring Failures**
10. **A10: Server-Side Request Forgery (SSRF)**
